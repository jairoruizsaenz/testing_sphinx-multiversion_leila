
:: Cree un entorno virtual e instale las librerías necesarias mediante el archivo requirements.txt o proceda a instalar las siguientes librerías.

pip install leila
pip install -U sphinx
pip install sphinx_rtd_theme
pip install sphinx_copybutton
pip install sphinx-multiversion

:: Se debe incluir el template de versiones y hacer en el enlace en el archivo cong.py para agregar la pestañas de versiones con el tema read the docs de sphinx

Plantilla de refenciar: https://holzhaus.github.io/sphinx-multiversion/master/templates.html#readthedocs-theme
Se modificó la plantilla para poder cambiar el nombre del branch master

----------------------------------------------------

sphinx-multiversion ./sphinx/source .
sphinx-multiversion ./sphinx/source . --dump-metadata
